# Pixel_Filter_JS
一個透過 Javascript 製作的像素濾鏡工具。

這個專案重構於我的 [像素濾鏡工具](https://github.com/JingShing/Pixel-Art-Filter-Web).

你可以將想要像素化的檔案放入 content 區。

調整好數值後。點擊按鈕，將自動像素化檔案。
